## 🛠️ How to Use Locally

```bash
# Clone the repo
git clone https://github.com/TechTropolis/techtropolis.git

# Go into the folder
cd techtropolis

# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
